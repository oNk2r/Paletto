# Paletto - Color Palette Generator
Color palette generator designed to quickly curate harmonious colors, test WCAG contrast ratios, and seamlessly export ready-to-use variables.

## ✨ Features

- **Intuitive UI**: Beautiful, fully responsive grid-based dark theme dashboard.
- **Color Formats**: Input and interact with colors via HEX, RGB, or HSL.
- **Smart Generation**: Automatically computes mathematical color variants (shades, complementary, analogous, triadic).
- **Accessibility Built-In**: Real-time WCAG contrast scoring against White and Black UI contexts to ensure accessible design implementations.
- **Instant Exporting**: Direct download option of your entire color suite to JSON.

## 🚀 Running Locally

1. Install module dependencies:
   ```bash
   npm install
   ```
2. Boot the development environment:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4
- **Icons & Polish**: Lucide React + React Hot Toast
